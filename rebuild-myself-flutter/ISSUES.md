# 产品优化清单

## P0 — 必须修

- [x] **修改密码是空壳** — 设置页 → 修改密码弹窗只有标题和按钮，没有输入框和实际逻辑
- [x] **同步数据按钮是假的** — 设置页 → 同步数据只弹假 SnackBar，没有调用 SyncService
- [x] **导出数据按钮是假的** — 设置页 → 导出数据只弹假 SnackBar，没有实际导出逻辑
- [x] **全局删除无确认** — 目标、任务、记录、计划等所有删除操作都没有"确定删除？"二次确认
- [x] **计划改动不同步到服务端数据库** — `reorderPlan`/`generateTodayPlan` 只存本地，`updatePlanContent`/`updatePlanTime` 调用的服务端接口不存在

## P1 — 该修

- [ ] **隐私锁密码不持久化** — 重启 app 后密码丢失，用户需重新设置
- [ ] **输入没有校验** — 情绪评分 1-10 可输入 999，耗时分钟数等也未校验
- [ ] **4 套独立计时器** — 首页焦点计时器、学习页、财务页、干预页各有独立 Timer.periodic，互不知晓，应统一用 FocusTimerProvider
- [ ] **仅浅色主题** — 设置页"主题"只显示"浅色"无切换，缺暗色模式

## P2 — 建议修

- [ ] **空闲感页面是死页面** — `/empty-mood` 路由已定义但 app 中无任何入口，功能模块白写了
- [ ] **首页统计用的是全局数据** — "今日记录"计数、情绪评分取的是全量数据而非当天数据
- [ ] **目标页离线不可用** — 其他模块有本地兜底，GoalProvider 只从 API 加载，断网时空白
- [ ] **AI 报告有两个入口** — 统计页（tab 3）和设置→AI报告（/reports）功能重叠，用户会困惑
- [ ] **休闲页替代活动不可点击** — "无聊时试试这些"6 个卡片纯装饰，点击无反应
- [ ] **因果链"阻断方案"按钮是空的** — EmptyMoodPage 的 `onPressed: () {}`

## P3 — 锦上添花

- [ ] **没有编辑功能** — 全 app 只能新增+删除，不能修改已有记录
- [ ] **字段命名不一致** — camelCase/snake_case 双写删除（`planDate` + `plan_date`），表明 DB 写入模式不统一
- [ ] **UI 细节不统一** — 进度条高度（3px/6px/8px）、标签圆角（3px/4px/6px）各处不同
- [ ] **干预记录触发描述未保存** — `_formTriggerCtrl.text` 获取了但没传给 BehaviorIntervene
- [ ] **干预页连续天数硬编码为 0** — 没有实际计算
